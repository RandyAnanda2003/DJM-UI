Apa itu DJM?

Distinct Job Manual (DJM) adalah dokumen standar yang digunakan di Telkom Indonesia untuk mendokumentasikan detail pekerjaan yang dilakukan oleh employee telkom. DJM mencatat informasi penting seperti:

Mission Statement

goals utama dari posisi employee

Job Responsibilities

Tanggung Jawab employee selama bekerja

Job Performance Index

Skala pengukuran performa employee

Job Authorities

Hak-Hak Employee yang diberikan untuk menunjang aktivitas selama bekerja

Dengan sistem automasi DJM ini, proses pembuatan dokumen menjadi lebih cepat, akurat, dan terstandarisasi, mengurangi kesalahan manual dan meningkatkan efisiensi operasional.

Cara Menggunakan DJM Automate

Ikuti langkah-langkah mudah berikut untuk membuat DJM Anda

Klik tombol "+ BUAT DJM" di halaman make DJM, lalu upload file Excel berisi formasi posisi dan file PDF Peraturan Organisasi yang sesuai dengan formasi.

  • • Format file: .xlsx atau .xls dan .pdf
  • • Pastikan kolom sesuai template standar playbook
  • • Pastikan untuk proses HCSM mengupload semua formasi posisi dari BP I-VI yang akan digenerate
  • • sekali file posisi/formasi diupload, DILARANG MENGUBAH JobID dan Objid untuk seluruh tahapan

Masukan nama organisasi atau unit kerja yang akan diproses. nama ini bisa menjadi identifier komunikasi antara unit HCSM dan HCSP.

Contoh organisasi:

Telkom Regional 1(Randy-16/07/25) Telkom Regional 1(Ical) Direktorat IT (Aulia)

Tentukan tahap proses yang sesuai dengan pekerjaan Anda:

HCSP

Human Capital Support Partner - akan menginput 3 hal, yaitu proses ID organisasi yang sama dengan HCSM (komunikasikan dengan HCSM), nama organisasi, dan File Excel tervalidasi oleh HCSM. keluaran untuk tim HCSP adalah DJM BP 1-6

HCSM

Human Capital Strategic Management - akan menginput 3 hal, yaitu Nama organisasi, File Excel Posisi sesuai aturan playbook, File PDF peraturan organisasi. keluaran untuk tim HCSM adalah DJM BP 1-3 Managerial.

Sistem akan otomatis memproses file Anda. Status akan ditampilkan secara real-time:

Sedang Berjalan - Proses sedang berlangsung
Selesai - File Excel hasil Generate siap untuk diunduh
Error - Terjadi kesalahan pada sistem atau input data, periksa kembali input data anda atau hubungi developer

⏱️ Waktu proses bervariasi tergantung ukuran data, umumnya 0.7 Menit per Posisi

Setelah status berubah menjadi "Selesai", klik tombol "Download Excel" untuk mengunduh hasil DJM Anda.

✅ File hasil berisi:

  • → Data DJM yang terstruktur dan terformat (Job ID, Objid, Nama Posisi, MS, JR, JPI, JA)
  • → Siap untuk divalidasi dan diperiksa secara manual
  • → Excel siap inject ke Jarvis, dapat diperoleh setelah proses validasi HCSP
